Special point plot files:

Most grayscale images supplied by Monty McGraw. The original files are from
Chuck Forsberg.

spock.plt and zz_scalemodel.plt were created by Rene Richarz using
https://github.com/rricharz/make-tek-image
They are using the very unlinear 6-bit grayscale of the Tektronix 4014.

The scale model of the Tektronix 4010 terminal displayed in zz_scalemodel.plt
was crafted by Dave Ault.
